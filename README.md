# 🧠 EEG Brainwave Art

**p5.js generative art** — real-time EEG brainwave visualization simulating calm-to-anxious state transitions.

Built for [NEURODRIVE](https://github.com/Rewsyaydee/fyp-neurodrive) FYP.

## 🎨 5 Frequency Bands

| Band | Hz | Color | State |
|------|-----|-------|-------|
| Delta | 0.5–4 | Blue | Deep sleep |
| Theta | 4–8 | Cyan | Drowsy |
| Alpha | 8–13 | Green | Relaxed |
| Beta | 13–30 | Orange | Alert |
| Gamma | 30–100 | Red | Anxious |

## ✨ Features
- Auto-cycling 45s calm→anxious transitions
- Mouse Y controls anxiety level
- Real-time anxiety meter + particle sparks
- Dark gradient shifts cool blue → warm red
